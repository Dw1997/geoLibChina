# lnglat2Geo

经纬度转省市区县，速度快 -- 单线程50000次/秒；精度高 -- 准确率100%

还包含如下功能：
1：查询某个经纬度附近所有商圈，按距离排序
2：给定城市，输出城市级别
3：输入任何地区的全称/简称/code，输出该地区的全部信息
4：获取所有行政区划关系数据等

使用方法：
import com.dengxq.lnglat2Geo.GeoTrans 里面的所有方法均为公有接口
